## 当前功能架构（代码事实）

- 后端：`backend/app/main.py` 提供 FastAPI 应用、`/health` 和允许 `FRONTEND_URL` 来源访问 API 的 CORS 配置；`backend/app/api/routes/auth.py` 提供注册、登录、当前用户和免注册 demo session 接口；`backend/app/api/routes/personas.py` 提供用户隔离的人物 list/create/detail/update/delete；`backend/app/api/routes/materials.py` 提供用户隔离的资料上传、手动资料、资料列表、详情、删除和重新排队接口；`backend/app/api/routes/jobs.py` 提供用户隔离的 AI Job 列表、详情、重试和取消接口；`backend/app/api/routes/memories.py` 提供用户隔离的记忆列表、创建、详情、编辑、确认、拒绝、禁用和删除接口，并在记忆审计动作后刷新 profile/trust；`backend/app/api/routes/profile.py` 提供用户隔离的 profile GET/PATCH/regenerate 和 recalculate-trust API；`backend/app/api/routes/chat.py` 提供用户隔离的 conversation list/create、message list/send、voice-message、message citations 和 correct-memory API；`backend/app/api/routes/voice.py` 提供用户隔离的 voice config、默认 TTS、音色样本、音色克隆和 mock 语音合成 API；`backend/app/api/routes/avatar.py` 提供用户隔离的 avatar config、默认形象和 mock 3D 生成 API；`backend/app/services/persona_prompt.py` 输出包含 `profile_summary` 的 `prompt_context`；`backend/app/services/materials.py` 写入本地 storage 并触发 mock parse jobs；`backend/app/services/parsing.py` 通过 `backend/app/providers/gateway.py` 的 deterministic mock capabilities 生成 `ParsedChunk` 和 source-backed `MemoryCard`；`backend/app/services/profile.py` 聚合 `PersonaProfile`、计算可解释可信度并记录显式重算 jobs；`backend/app/services/chat.py` 使用 deterministic retrieval、mock `chat_llm` Provider Gateway 输出、对话历史、`message_citations`、mock `asr` AI Job 和 TTS audio URL 生成第一人称文本/语音消息回复；`backend/app/services/voice.py` 使用 `VoiceModel`、mock `tts`/`extract_voice_sample`/`voice_clone` Provider Gateway、`synthesize_speech`/`extract_voice_sample`/`clone_voice` AI Job 生成 mock audio URL、sample_ready/cloned_ready/clone_failed 状态和默认 TTS fallback；`backend/app/services/avatar.py` 使用 `AvatarModel`、mock `avatar_3d` Provider Gateway 和 `avatar_3d` AI Job 生成 default_avatar/generated_ready/generation_failed 状态、GLB mock URL、预览图、基础动作和口型配置；`backend/app/models/`、`backend/alembic/` 和 `backend/tests/` 提供 SQLAlchemy 模型、初始迁移与测试。
- 前端：`frontend/app/` 提供中文 Next.js 页面和人物工作流，并已新增 `/personas/{id}/uploads`、`/personas/{id}/jobs`、`/personas/{id}/memories`、`/personas/{id}/profile` 与 `/personas/{id}/chat` 页面；`frontend/src/components/DemoEntry.tsx` 提供免注册演示入口；`frontend/src/lib/api.ts`、`frontend/src/lib/auth.ts`、`frontend/src/lib/persona.ts`、`frontend/src/lib/materials.ts`、`frontend/src/lib/jobs.ts`、`frontend/src/lib/memories.ts`、`frontend/src/lib/profile.ts`、`frontend/src/lib/chat.ts` 和 `frontend/src/lib/routes.ts` 提供 API 路径、JWT/demo helper、人物/资料/任务/记忆/profile/chat 数据 helper 和路由常量；`frontend/tests/auth.test.mjs`、`frontend/tests/routes.test.mjs`、`frontend/tests/persona.test.mjs`、`frontend/tests/materials.test.mjs`、`frontend/tests/memories.test.mjs`、`frontend/tests/profile.test.mjs` 与 `frontend/tests/chat.test.mjs` 覆盖 demo helper、基础路由、人物表单校验、资料/任务 helper、记忆枚举/helper、profile helper、chat helper 和 trust level contract。
- 集成：`backend/Dockerfile`、`backend/docker-entrypoint.sh`、`backend/.dockerignore`、`frontend/Dockerfile`、`frontend/.dockerignore`、`docker-compose.yml` 和 `docs/init.sh` 已接入统一验证路径。
